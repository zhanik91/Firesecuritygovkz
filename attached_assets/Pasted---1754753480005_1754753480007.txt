2.2 Требования к размещению и установке огнетушителей
Высота и способы установки: Переносные огнетушители массой до ~15 кг обычно размещаются на вертикальных поверхностях (стенах, колоннах) или в пожарных шкафах. ППБ требуют, чтобы верхняя часть огнетушителя находилась на высоте не более 1,5 м от пола, а расстояние от пола до нижней части – не менее ~0,15 м (для защиты от сырости)
nfcom.ru
. Более тяжёлые огнетушители (>15–20 кг) допустимо устанавливать на полу, в специально отведённых местах или катать на тележке. В зданиях зачастую используют комбинированные шкафы (ПК) – внутри каждого пожарного шкафа должно размещаться не менее двух ручных огнетушителей 5 кг (помимо пожарного крана). Доступность и видимость: Огнетушители следует располагать в легкодоступных и заметных местах – возле выходов из помещений, на путях эвакуации, у входов в здание. Обязательно должна быть табличка или знак указания места огнетушителя. При размещении в нишах или шкафах – дверца шкафа маркируется соответствующим символом. Расстояние от предполагаемого очага пожара до ближайшего огнетушителя не должно превышать нормативных значений: согласно приложению 3 ППБ, это 20 м – для общественных зданий, 30 м – для помещений категорий А, Б, В (высокая опасность), 40 м – для категории Г, 70 м – для категории Д
nfcom.ru
. Эти расстояния учитываются при распределении огнетушителей по площади: например, в большом производственном цехе их нужно разместить так, чтобы любой участок цеха находился в радиусе ~30 м от какого-либо огнетушителя. Исключения (где огнетушители не устанавливаются): Есть ситуации, когда размещение огнетушителей не только не требуется, но и запрещается по соображениям безопасности. Например, в камерах следственных изоляторов и тюрем первичные средства пожаротушения не устанавливаются – во избежание того, чтобы ими завладели заключённые; при пожаре в таких помещениях тушение обеспечивают дежурные службы извне. Другой пример: помещения категории Д ≤100 м² – могут не иметь огнетушителей (см. п.2.1). Также в некоторых высокоавтоматизированных зонах (электрошкафы с газовым тушением и т.п.) ручные огнетушители могут не требоваться, если риск минимизирован. Но в целом, во всех общественных и производственных помещениях наличие ПСП строго обязательно.
2.3 Выбор типа и массы огнетушителей
Класс пожара и огнетушащее вещество: Огнетушитель должен быть способен тушить наиболее вероятный тип возгорания в данном помещении. Руководствуются следующим:
Класс A (твердые горючие материалы: дерево, бумага, ткань) – эффективны водные и пенные огнетушители (ОВП – воздушно-пенные), а также универсальные порошковые. CO₂ не так эффективен для тлеющих материалов.
Класс B (горючие жидкости, ГСМ) – нужны порошковые (АВС) либо углекислотные (ОУ) огнетушители. Пенные тоже годятся для нефти/растворителей, но не применимы на электроустановках.
Класс C (газы) – при горении газа применяют порошок, либо сначала перекрывают подачу. Специально класс C покрывает порошковый АВС.
Класс D (металлы) – только специальные порошковые (например, на основе хлорида натрия, графитовые). Это редкие случаи, для лабораторий с натрием и т.п.
Класс E (электрооборудование под напряжением) – углекислотные (ОУ) наиболее предпочтительны (не портят электронику, диоксид углерода не проводит ток)
dogovor24.kz
. Порошковые тоже можно (класс Е есть у большинства АВС порошков), но мелкий порошок может повредить технику. Хладоновые огнетушители (фторированные газы) оптимальны для серверных, музеев, архивов – они не оставляют следов и безопасны для оборудования
nfcom.ru
. Водные и пенные огнетушители запрещены для класса E, т.к. проводят электрический ток.
Обычно применяют порошковые огнетушители универсального типа (А,В,С,Е) массой 5–10 кг как основу защиты большинства помещений – из-за их широкого спектра применения и морозостойкости. Дополнительно, в электрощитовых и серверных – углекислотные ОУ-5, ОУ-8. В кухнях (жиры) могут использоваться аэрозольные или специальные (класс F). В общем, вид огнетушителя определяется классами пожара, которые вероятны в помещении
dogovor24.kz
. Масса заряда и ранги огнетушителей: Каждый огнетушитель имеет показатель огнетушащей способности – условный ранг (например, 2A, 55B и т.д. в европейской классификации). От массы заряда зависит площадь, которую он может потушить. Для простоты, нормативы устанавливают минимальные массы заряда: так, в пожарных шкафах требуются >=5 кг каждый. Типовые переносные огнетушители: 2 кг (офис, автомобиль), 5 кг (универсальный для комнаты ~50–100 м²), 9–10 кг (для складов, цехов). Передвижные огнетушители на тележках имеют массу заряда 20–100 кг (например, ОП-20, ОП-50, ОУ-80). Их применяют там, где потенциальный очаг велик по площади и маленькие баллоны неэффективны. В правилах указано: если возможен значительный по площади очаг (более 1 м²), необходимо иметь передвижные огнетушители
integral-d.ru
integral-d.ru
. Практически, этот критерий означает: склады и производственные здания площадью >500 м² – обязателен хотя бы один передвижной огнетушитель (или стационарное средство тушения). Резервные огнетушители и перезарядка: Согласно СТ РК 1487, на объекте должен предусматриваться резерв заряженных огнетушителей на случай вывода рабочих ОТ из эксплуатации. Если огнетушитель отправлен на перезарядку или ремонт – на его место сразу ставится резервный аналогичной мощности
zakup.sk.kz
. Это означает, что организация должна иметь некоторое количество запасных огнетушителей (не меньше ~10% от парка, либо по одному на каждую группу) для подмены. Сами огнетушители подлежат регулярной проверке и перезарядке по графику: ежегодно проводится внешний осмотр и (для некоторых) выборочная проверка массы; раз в 5 лет – обязательная полная перезарядка порошковых, углекислотных и хладоновых огнетушителей
zakup.sk.kz
. Водные и пенные перезаряжаются чаще – ежегодно (пену на основе пенообразователя – раз в 1–2 года)
zakup.sk.kz
zakup.sk.kz
. График проверок сводится в журнале технического обслуживания. Таким образом, при расчёте количества ОТ нужно учесть и резерв на время ТО: например, если в здании 10 огнетушителей, желательно иметь ~2 запасных, чтобы при поочередной перезарядке поддерживать норму оснащённости.
2.4 Особые случаи снижения или увеличения норм
Правила предусматривают корректировки норм в отдельных случаях:
Помещения, защищённые АУПТ (автоматической установкой пожаротушения) – могут оснащаться ручными огнетушителями лишь на 50% от расчетного количества
nfcom.ru
. То есть, если по площади положено, условно, 4 шт., то при наличии спринклеров допустимо установить 2 шт. Данное послабление объясняется тем, что автоматическое тушение значительно снижает вероятность развития пожара, и большое число ручных средств может не понадобиться. Важно: снижение на 50% – это право, а не обязанность; многие объекты, несмотря на спринклеры, оставляют полный комплект огнетушителей для подстраховки. Также минимум (2 на этаж) все равно должен соблюдаться.
Сейсмически опасные зоны – (в нормативах соседних стран, напр., РБ) встречается требование увеличивать количество огнетушителей на 20% в сейсмических районах. В казахстанских ППБ прямо это не указано, но проектировщики могут учитывать местные условия.
Необорудованные водой территории – согласно п.21 прил.3 ППБ, если в производственных/складских помещениях отсутствует внутренний водопровод и объект удалён от наружных водоисточников >100 м, то требуются пожарные щиты с набором инвентаря
nfcom.ru
. Это косвенно увеличивает объем первичных средств (дополнительно к огнетушителям – щит с лопатами, баграми, кошмой, ведрами). Количество щитов определяется по площади и категории, напр. 1 щит на каждые 500–1000 м² территории, в зависимости от категории риска
dogovor24.kz
dogovor24.kz
.
Противопожарный режим объекта: Если на объекте организовано круглосуточное дежурство пожарной команды (как в крупных нефтехранилищах), то допускается часть переносных средств заменить стационарными постами с порошковыми модулями или доверить функцию первой помощи самой команде. Однако формально ПСП должны присутствовать в требуемом объёме всегда.
Взрывопожароопасные зоны: В местах, где хранение огнетушителя может быть опасно (например, внутри взрывоопасной зоны категории А, где даже удар о металлический баллон может вызвать искру), могут устанавливаться стационарные модули тушения вместо ручных, а переносные ОТ размещаются вне зоны (на входе). Это решается проектно с учетом требований технического регламента по взрывозащите.
Подытоживая, алгоритм подбора огнетушителей для конкретного помещения следующий: Шаг 1: Определить категорию помещения (А, Б, В, Г, Д) и класс возможного пожара (A, B, C, D, E) – на основе технологического процесса, хранения веществ и функционального назначения
online.zakon.kz
online.zakon.kz
. Шаг 2: По категории и площади найти базовую норму: кол-во огнетушителей = (площадь / предельная площадь на 1 ОТ) с округлением вверх. Предельные площади берутся из таблиц норм (см. п.2.1). Учитывать, что на каждый этаж ≥2 шт. даже при малой площади

. Шаг 3: Проверить особые условия: есть ли АУПТ (если да – можно разделить результат на 2

nfcom.ru
), нет ли запретов (как в СИЗО). Шаг 4: Выбрать тип и массу огнетушителей: исходя из классов пожара и температур. Обычно стараются универсальными порошковыми закрыть класс A, B, C; если есть электрооборудование – включить хотя бы один CO₂. Массу заряда брать максимальную возможную, которую удобно применять в данном помещении (например, 5–9 кг для комнаты, 9–15 кг для склада). Проверить, нужна ли передвижная тележка (если общая площадь >500 м² или есть угроза большого разлива ЛВЖ – нужна 100%). Шаг 5: Определить места установки: равномерно по площади, у основных эвакуационных выходов, на видимых местах. Соблюсти расстояния (20/30/40/70 м – по категории)
nfcom.ru
 и высоту ≤1,5 м
nfcom.ru
. Прикрепить или поместить в шкафы, опломбировать (пусковые рычаги пломбируются). Шаг 6: Назначить ответственное лицо за эксплуатацию огнетушителей (по п.9 ППБ – приказом руководителя назначается ответственный за приобретение, сохранность и готовность ПСП). Организовать журнал учета проверок и своевременную перезарядку по графику стандарта. Предусмотреть обменный фонд (резерв) на время перезарядки
zakup.sk.kz
. Выполнив эти шаги, объект будет обеспечен первичными средствами пожаротушения в соответствии с нормами. Далее рассмотрим требования по созданию негосударственной противопожарной службы (НГПС) – когда она нужна, сколько техники и людей должно быть.
3. Расчёт требуемых техники и личного состава для НГПС
Негосударственная противопожарная служба – это ведомственные (частные) пожарные подразделения, создаваемые предприятиями для защиты своих объектов. Закон РК «О гражданской защите» (ст.53) устанавливает, что перечень объектов, на которых обязательно создается НГПС, утверждается правительством

. Такой перечень существует (Постановление Правительства №1017 от 25.09.2014 с изм. 2015) – он содержит отрасли и критерии. Кроме того, сама НГПС должна соответствовать ряду нормативов: иметь аттестацию, обученный персонал, пожарную технику и снаряжение по нормам не ниже государственных. Ниже приведена методика определения, нужно ли предприятию создавать НГПС, а если да – сколько пожарной техники и людей требуется, с обоснованием.
3.1 Обязательность создания НГПС: критерии и категории объектов
Согласно упомянутому перечню, НГПС в обязательном порядке создаётся на следующих объектах (выдержки из Приложения, с примерами):
🔥 Особо пожаро- и взрывоопасные производства: предприятия нефтегазодобычи и переработки, нефте- и газохимические заводы, объекты химической промышленности – обязаны иметь собственные пожарные службы. (Например, нефтеперерабатывающий завод, химический комбинат). Эти объекты включены прямым пунктом перечня
law.gov.kz
 независимо от удаленности от города, т.к. риск высок.
🔥 Энергетические объекты крупной мощности: вероятно в перечне присутствуют большие электростанции (ТЭЦ, ГЭС) и высоковольтные подстанции – по значимости они требуют НГПС.
🔥 Объекты с массовым пребыванием людей высокой этажности: конкретно в перечне 2014 не указаны, но по практике – например, международные аэропорты, крупные метрополитены, возможно, тоже должны иметь свои пожарные подразделения.

online.zakon.kz
online.zakon.kz
. То есть, если ваш завод крупный и находится далеко от ближайшей городской пожарной части (>3 км), вам необходимо организовать свою пожарную команду с пожарными автомобилями. Оба условия, как разъяснил МЧС, применяются совместно (и площадь, и расстояние) – меньший объект вдали или большой объект в городе по этому пункту не подпадают.
🔥 Прочие объекты из перечня: водохранилища и гидротехнические сооружения (плотины), особо ценные объекты культуры, уникальные производства – перечислены правительством отдельно. Их относительно немного, но они есть.
Если объект не входит в обязательный перечень, то создавать свою пожарную службу не требуется. Пожарную безопасность обеспечивает государственная противопожарная служба (ГПС) – т.е. обычные городские/районные пожарные части МЧС. Однако закон допускает альтернативу: даже при отсутствии прямой обязанности предприятие может заключить договор с существующей НГПС (частной пожарной организацией) на обслуживание
edu.kamkorservice.kz
. Например, на охраняемой территории несколько заводов могут совместно содержать одно пожарное депо или нанять аккредитованную частную пожарную команду. Такой договор учитывается как выполнение требований (п.3 ст.53 Закона)
edu.kamkorservice.kz
. Алгоритм проверки необходимости НГПС:
Проверить отрасль и назначение объекта против правительственного перечня. Если отрасль попадает (нефтегаз, химия, энергетика и т.п.) – НГПС нужна безусловно.
Если отрасли нет в прямом списке, проверить пожароопасность и размеры: категория помещений (А, Б или В1–В4) и суммарная площадь таких зданий. Если ≥3500 м² и объект удалён (>3 км) от ближайшей пожарной части МЧС – НГПС требуется (с пожарной техникой)
online.zakon.kz
.
Если объект поменьше или ближе к городу – собственная служба не обязательна. Но по желанию можно создать пожарный пост без техники (см. ниже), либо обеспечить усиленные ПСП и обучение персонала. Многие предприятия организуют добровольные пожарные дружины, но это не НГПС в строгом смысле.
Таким образом, обоснование для НГПС – либо особо высокий риск, либо большое удаление и время прибытия ГПС. В проектах категорий риска 1 или 2 (в соответствии с методикой оценки) это отражается и обычно предписывается создание ведомственной пожарной охраны.
3.2 Виды подразделений НГПС: пожарная часть vs пожарный пост
Нормативно различаются два основных формата негосударственных подразделений пожаротушения
adilet.zan.kz
adilet.zan.kz
:
Пожарная часть (ПЧ) – с выездной пожарной техникой. Имеет на вооружении основные и специальные пожарные автомобили, расчёты, депо для их размещения. Пожарная часть НГПС организуется там, где по требованиям нужны собственные пожарные машины для оперативного выезда на возгорание на территории предприятия.
Пожарный пост – без выездной техники. Как правило, это круглосуточный пост дежурных пожарных на объекте, оснащённых только переносными средствами (огнетушителями, мотопомпой, пож. инвентарём) и связью. Пост может патрулировать территорию, следить за системами сигнализации и в случае пожара приступать к тушению подручными средствами, ожидая прибытия государственной пожарной охраны. Пожарные посты допускаются на менее опасных объектах, где наличие техники необязательно (например, небольшое предприятие близко к городу, но руководство желает иметь своих пожарных для первой реакции).
Пример: если ваш объект – складской комплекс 4000 м², 5 км от города, по критериям обязателен свой пожарный с выездной техникой, то есть полноценная пожарная часть (минимум с одной автоцистерной). Если же комплекс 1000 м² в 10 км от города – формально перечень его не охватывает, но вы обеспокоены рисками – можно создать пожарный пост (2–3 человека с автомобилем-вездеходом и мотопомпой, условно). Соответствие оснащения: Закон требует, чтобы НГПС по материально-техническому оснащению не уступала государственной
edu.kamkorservice.kz
. Это означает: наличествующая техника, оборудование, снаряжение и боевое обмундирование должны соответствовать нормам положенности, принятым в МЧС. Например, на каждого пожарного – каска, боёвка, сапоги, самоспасатель; на каждую машину – пожарно-техническое вооружение (рукава, стволы, бензоинструмент и пр.) по нормам. Работники НГПС обеспечиваются спецобмундированием и снаряжением по нормам ГПС
edu.kamkorservice.kz
, а сама НГПС – пожарной техникой и вооружением по нормам ГПС
edu.kamkorservice.kz
. Фактически, ведомственная часть оснащается как обычная пожарная часть МЧС.
3.3 Нормы укомплектованности: техника и расчёты
Основная пожарная техника: К основным видам пожарных машин, которые могут потребоваться предприятию, относятся:
Пожарные автоцистерны (АС, АЦ) – основная машина первой помощи, возит воду/пену. Бывает разной ёмкости (2,5 т, 5 т, 10 т). Как правило, хотя бы одна автоцистерна должна быть в каждой пожарной части НГПС. На крупных объектах – несколько. Например, на нефтебазе с большим запасом ЛВЖ может требоваться 2 автоцистерны и 1 пенная автоцистерна.
Автолестницы или коленчатые подъёмники – нужны на объектах с высотными сооружениями (эстакады, колонны, здания выше ~15 м). Если на предприятии есть высокие конструкции, НГПС может комплектоваться подъёмной техникой для спасения и подачи стволов на высоту.
Специальные пожарные автомобили: к ним относят автонасосы высоконапорные, пенообразователи, порошковые автомобили, аварийно-спасательные автомобили (для ДТП, техногенных ЧС), дымоотводные машины и т.д. Выбор зависит от специфики. Например, на химическом производстве может быть пожарный порошковый автомобиль, на аэродроме – специальная автоцистерна повышенной проходимости, на шахте – специализированный автомобиль ГЗДС.
Передвижные пожарные насосы, мотопомпы – иногда вместо второй автоцистерны можно иметь передвижную мотопомпу с производительностью не менее основной. Это применимо для объектов с открытыми водоёмами.
Штатная численность караулов: В нормативных документах прямо указываются минимальные расчёты на технику. Стандартно, один пожарный автомобиль обслуживается боевым расчётом из 3–4 человек (1 водитель и 2-3 пожарных). Например, типовой караул ГПС на автоцистерне – 1 командир отделения, 1 водитель, 2 пожарных (итого 4). На автолестницу – 1 водитель и 1 пожарный (2 чел.). На аварийно-спасательный авто – 2-3 чел. Для НГПС эти цифры аналогичны. Минимально в смену должно заступать не менее 2 человек (даже на пожарном посту без техники). Если есть 1 пожарная машина – обычно дежурит 4 человека в смену (как выше). Если 2 машины – 6-8 человек. Кроме того, нужна старшая смены (начальник караула). На крупных объектах вводятся диспетчеры, связи и пр. Таким образом, расчёт численности производится исходя из числа единиц техники и принятого графика дежурств. Как правило, пожарные работают в суточном режиме (24 часа через 3 дня, т.е. 3-4 смены). Значит, штат на одну должность = 4 человека, чтобы покрыть круглосуточно. Например, одна должность “пожарный” в расчёте = 4 человека в штате (четверо сменяют друг друга). Поэтому если в одном карауле 4 человека, для круглосуточного обеспечения требуется 4×4 = 16 человек личного состава (плюс начальники). Для упрощения: на каждый пожарный автомобиль нужно ~12–15 человек штатно, чтобы покрыть три или четыре караула и отпускные графики. Приведём небольшой ориентир в виде таблицы:

3.4 Классификация пожарной техники и оснащение НГПС
При формировании НГПС важно правильно подобрать тип пожарной техники, исходя из характеристик объекта:
Основные пожарные автомобили:
Автоцистерна (АЦ) – основной вид, предназначен для подачи воды/пены. Обозначается, например, АЦ-40(130) – насос 40 л/с, цистерна на шасси ЗИЛ-130. В НГПС часто применяют заводские пожарные машины или приспособленные (например, на базе КАМАЗ). Нормы ГПС требуют обеспечение запаса воды не менее 3–4 тонн на выезд.
Автонасос высокого давления – спецвариант автоцистерны с меньшим запасом воды, но мощным насосом (например, для тушения резервуаров через лафет).
Автолестница (АЛ) – для подъёма на высоту, актуально при высоких сооружениях или эстакадах. Маркируется высотой, напр. АЛ-30 (30 м). Если на объекте нет высотных конструкций – можно не иметь.
Автоподъёмник (коленчатый, АКП) – альтернатива лестнице, для сложного рельефа.
Специальные и вспомогательные машины:
Пенное пожарное авто (АП) – оснащено большим запасом пенообразователя и генераторами пены. Для нефтебаз – почти обязательны, т.к. тушение нефти водой малоэффективно.
Порошковый пожарный автомобиль – несёт тонны порошка, актуален на газовых объектах, электрических подстанциях (где вода нежелательна).
Газодымозащитная служба (автомобиль ГДЗС) – может быть просто комплектами дыхательных аппаратов на основной машине, либо отдельный штабной автомобиль с запасом баллонов, если объект подземный, например.
Аварийно-спасательный автомобиль (АСА) – для техногенных аварий (ДТП на территории предприятия, завалы). Носит гидравлический инструмент, осветительные мачты и пр. Может быть полезен на крупных промпредприятиях.
Штабной автомобиль – для руководства тушением, на очень крупных объектах (НПЗ) целесообразно иметь, но чаще функции штаба выполняет автомобиль начальника НГПС.
Прицепы и мотопомпы: довольно часто НГПС оснащают переносными мотопомпами или насосными станциями, особенно если на объекте есть открытые водоёмы или резервуары. Например, при пожаре в дальнем конце объекта можно бросить мотопомпу к реке, вместо того чтобы ездить автоцистерне.
Оснащение пожарного депо НГПС: должно включать помещения для техники (бокс(ы) с выездом), диспетчерскую, комнату дежурных, склад СОЖ и ПТВ (средств огнетушения и вооружения). Местные исполнительные органы устанавливают требования к пожарным депо (как и для ГПС). Обычно, депо ведомственной части проходит инспекцию при аттестации.
3.5 Обоснование количества техники НГПС
Количество единиц техники определяется, исходя из необходимого расчётного боевого развертывания при возможном крупнейшем пожаре на объекте. Иными словами: сколько пожарных стволов и какого расхода потребуется. Например, для резервуара нефти диаметром 40 м по расчетам нужно подать 4 ствола ГПС-600 (пенных) – это под силу 2 пенным автоцистернам. Значит, нужно минимум 2. Если есть вероятность одновременного пожара в двух местах – число удваивается или предусматривается резерв. Однако на практике для обязательной НГПС минимумы таковы:
Объект базового уровня опасности (из критериев): минимум 1 пожарный автомобиль. Перечень №1017 прямо указывает: предприятия промышленности (≥3500 м², >3 км) – с выездной техникой (ед.ч.), не уточняя сколько машин, но подразумевается хотя бы одна АЦ
online.zakon.kz
. В реальных предписаниях МЧС зачастую пишут: “создать пожарное формирование, укомплектованное пожарным автомобилем”.
Особо опасный объект: как правило, не менее 2 единиц техники. Например, для нефтехимического завода МЧС обычно требует минимум 2 автоцистерны, а лучше 3–4 (с учётом резерва). Некоторые отраслевые нормы (приложение 10 к ППБ для нефтепродуктообеспечения) приводят перечень ПСП: там прямо указано, сколько автоцистерн, мотопомп и др. на какой объем резервуаров.
Если объект разбросан и большой – может потребоваться несколько пожарных постов или частей. Например, протяженный трубопровод охраняют несколькими пожарными поездами/постами на ключевых точках.
С учётом резерва: техника должна быть работоспособна всегда. Поэтому, если есть лишь одна автоцистерна и она выходит из строя – объект остаётся без защиты. На критически важных предприятиях желательно иметь резервный пожарный автомобиль или договор о помощи с соседними подразделениями. Нормативно на каждый выезд штатно один авто, но инженерная оценка риска может указать на потребность второго резерва.
Рекомендация: после идентификации самых опасных пожаров на вашем объекте, убедитесь, что суммарные возможности имеющейся техники НГПС >= потребностей. Например, максимальныйScenario – пожар в самом большом складе, нужен расход воды 20 л/с: одна стандартная автоцистерна (10 л/с) не справится, нужно 2 или доп. насос. Тогда закладывать 2.

Все пожарные прошли профессиональную подготовку (пожарно-технический минимум для работников противопожарной службы, обучение ГДЗС и т.д.). На работу в НГПС принимаются совершеннолетние граждане, прошедшие специальное обучение

.
Укомплектован штат, утверждены должностные инструкции, графики дежурств.
Имеется необходимая материальная часть (техника, оборудование) и исправна.
Разработаны оперативные документы: карточки тушения, планы взаимодействия с местной пожарной частью МЧС, схемы связи. Например, оперативный план пожаротушения должен быть на каждом энергетическом объекте.
Соблюдаются нормы трудовой безопасности (пожарные обеспечены СИЗ, медосмотр и т.д.).

4.2 Калькулятор определения необходимой НГПС
Этот инструмент предназначен для ответов на вопросы: нужно ли предприятию создавать свою пожарную службу? Если да, то какого типа (пост или часть) и сколько машин/людей планировать? Входные параметры:
Вид/отрасль объекта – выпадающий список (например: нефтяной/газовый объект, химическое производство, склад ГСМ, энергетика, прочее производство, административный, жилой и т.д.). Пользователь выбирает наиболее подходящее.
Площадь пожароопасных помещений, м² – числовое поле.
Категория этих помещений – (A, Б, В…).
Расстояние до ближайшей пожарной части МЧС, км – числовое поле.
Число людей на объекте (опционально, для информации).
Наличие уже договоров с пожарной охраной (опция, но скорее нет).
Логика расчёта:
На основе вида/отрасли сразу можно установить флаг mandatory. Например, если выбрано "нефтегаз" или "химическое производство" – обязательно НГПС (прямо по перечню) – тип пожарная часть. Если "прочее производство" – тогда проверять критерии площади и расстояния.
Проверить критерий: если категория ∈ {А, Б, В} и площадь >= 3500 и расстояние > 3 – вывод: НГПС с выездной техникой требуется (т.е. пожарная часть)
online.zakon.kz
online.zakon.kz
. Если расстояние >3, но площадь меньше – можно вывести: "формально не обязателен отдельный пост, но рассмотрите пожарный пост или договор, т.к. далеко от пожарных". Если площадь >=3500 но расстояние <=3 – формально не обязателен, но "при таком объёме производства, учитывая близость ПЧ, рекомендуется заключить договор с МЧС или иметь добровольную дружину".
Если вид объекта административный, жилой, торговый – как правило, НГПС не нужна, достаточно штатных ПСП и сигнализации. Скрипт может сразу выдавать "Нет, создание НГПС не требуется для объектов данного типа (достаточно охрана+МЧС)".
Определение типа подразделения:
Если выводится, что НГПС нужна, дальше логика:
Если mandatory по списку или площадь очень большая (>10000) – рекомендовать пожарная часть с минимум 2 авто.
Если по минимуму (только один критерий 3500/3км) – достаточно пожарная часть с 1 автоцистерной.
Если находится на грани (например, 3000 м², 5 км) – формально не обязано, но можно предложить "Пожарный пост без техники".
Рассчитать примерный состав:
Количество авто:
По умолчанию 1 (если нужна ПЧ).
Если отрасль нефтегаз/химия – ставим 2 (или больше, если очень большой объект, можно по площади градацию: >10000 м² – 2 авто, >20000 – 3 авто...). Такие градации не из нормативки прямо, а из логики.

  let type = document.getElementById('objType').value;
  let area = parseFloat(document.getElementById('objArea').value)||0;
  let cat = document.getElementById('objCat').value;
  let dist = parseFloat(document.getElementById('objDist').value)||0;
  let need = false, post = false, numAuto = 0, recommendation='';
  // Determine need:
  if(mandatoryList.includes(type)){
    need = true;
    numAuto = techRecommend[type] || 1;
  } else if(cat.match(/A|Б|В/) && area >= 3500 && dist > 3) {
    need = true;
    numAuto = 1;
  } else if(dist > 3 && area < 3500) {
    // Far but small
    recommendation += "Не обязательно, но расстояние большое – рассмотрите пожарный пост.\n";
    post = true;
  }
  // Determine vehicles:
  if(need){
    if(numAuto < 1) numAuto = 1;
    // Increase numAuto if area huge:
    if(area > 10000 && numAuto < 2) numAuto = 2;

Логика калькулятора огнетушителей (нормы РК):
Ввод: площадь (м²), категория (A/Б/В/Г/Д), этажи, флажок «есть АУПТ», флажок «есть электрооборудование».
Расчёт:

Базовая плотность (м² на 1 ОТ): A=50, Б=50, В=100, Г=200, Д=300.

Кат. Д и S≤100 → 0 шт.

Минимум 2 шт. на этаж.

При АУПТ → 50% от рассчитанного (но не ниже минимума).

Порог передвижного ОТ: если S>500 → добавить рекомендацию «ОП-25/50 на колёсах».

Типы: если «электро» → часть CO₂ (ОУ-5), базово — порошковый (ОП‑5/9).
Вывод: количество, типы, подсказки с якорными ссылками:
(#ppa3p8 – минимум 2 на этаж; #ppa3p13 – -50% при АУПТ; #ppa3p14 – расстояния; #st1487p85 – резерв).

Логика калькулятора НГПС:
Ввод: отрасль (select), площадь пожароопасных помещений (м²), категория (A/Б/В/Г/Д), расстояние до ближайшей ПЧ (км).
Правило обязательности:

Если отрасль ∈ {нефтегаз, химия, НПЗ, крупная энергетика} → НГПС обязательна (тип «пожарная часть»).

Иначе, если (категория ∈ {A,Б,В} и S ≥ 3500 и D > 3 км) → НГПС обязательна (ПЧ).

Иначе: НГПС не обязательна; если D>3 → рекомендовать пожарный пост.
Комплектация (ориентир):

ПЧ на 1 авто → штат 15–18 чел; 2 авто → 24–30 чел; 3 авто → 40+ чел.

Для нефтегаз/химии: рекомендовать 2 авто (АЦ + пенная/порошковая).
Вывод: «обязательна/не обязательна», рекомендуемая техника и диапазон штата. Ссылочные якоря: #pp1017p8, #z53p5.

UI/UX:

Поля ввода в карточках; кнопка «Рассчитать»; блок «Обоснование» с пунктами и якорями.

Tooltips на значках ⓘ, открывают краткий текст нормы.

Тест-кейсы (заложить в комментарии):

Офис 250 м², В, без АУПТ → 3 ОП‑5; расстояние ≤20 м.

Склад Б 600 м² → ~12 ОТ + 1 ОП‑50; без АУПТ.

Цех В 1200 м² с АУПТ → базово 12, с коэффициентом 50% → 6 шт., но ≥2/этаж.

НГПС: «прочее производство», 4000 м², В, 5 км → НГПС обязательна (1 АЦ, 15–18 чел).

// ===== Константы норм ППБ/СТ РК =====
const AREA_PER_OT = { 'A': 50, 'Б': 50, 'В': 100, 'Г': 200, 'Д': 300 }; // м² на 1 ОТ
const MIN_OT_PER_FLOOR = 2;
const THRESHOLD_WHEELED = 500; // м²
const HAS_AUTOPUT_FACTOR = 0.5; // -50%

// Рекомендации по типам
function recommendTypes(hasElec, cat, qty) {
  const parts = [];
  if (hasElec) parts.push(`из них 1–2 шт. CO₂ (ОУ‑5) для электро`);
  if (cat === 'A' || cat === 'Б' || cat === 'В') parts.push(`основа — порошковые ОП‑5/ОП‑9 (ABC)`);
  if (cat === 'Г' || cat === 'Д') parts.push(`подойдут ОП‑5 (универсальные)`);
  return parts.length ? `Рекомендуемые типы: ${parts.join('; ')}.` : '';
}

// === Огнетушители ===
window.calcExtinguishers = function () {
  const S = +document.getElementById('ext-area').value || 0;
  const cat = document.getElementById('ext-cat').value;
  const floors = Math.max(1, +document.getElementById('ext-floors').value || 1);
  const hasAUTP = document.getElementById('ext-autoput').checked;
  const hasElec = document.getElementById('ext-elec').checked;

  let qtyBase = 0;

  // Особый случай: категория Д ≤ 100 м² → можно не устанавливать
  if (cat === 'Д' && S <= 100) {
    document.getElementById('ext-result').innerHTML =
      `<p><b>Требуемое количество:</b> 0 шт.</p>
       <p>Допустимо не устанавливать (кат. Д ≤ 100 м²). <a href="#ppa3p9">См. примечание</a>.</p>`;
    return;
  }

  const areaPerOT = AREA_PER_OT[cat] || 100;
  qtyBase = Math.ceil(S / areaPerOT);

  // Минимум 2 на этаж
  const minByFloor = MIN_OT_PER_FLOOR * floors;
  let qty = Math.max(qtyBase, minByFloor);

  // Снижение при наличии АУПТ
  if (hasAUTP) {
    qty = Math.max(Math.ceil(qty * HAS_AUTOPUT_FACTOR), minByFloor); // не ниже минимума
  }

  // Рекомендация по передвижным
  const wheeled = S > THRESHOLD_WHEELED ? `Также рекомендуется передвижной ОП‑25/ОП‑50 (тележка) для больших очагов.` : ``;

  const types = recommendTypes(hasElec, cat, qty);

  const justification = [
    `<a href="#ppa3p8">минимум 2 на этаж</a>`,
    hasAUTP ? `<a href="#ppa3p13">−50% при АУПТ</a>` : null,
    `<a href="#ppa3p14">нормы расстояний</a>`,
    `<a href="#st1487p85">резерв на время ТО</a>`
  ].filter(Boolean).join(' • ');

  document.getElementById('ext-result').innerHTML =
    `<p><b>Требуемое количество:</b> ${qty} шт.</p>
     <p>${types} ${wheeled}</p>
     <p class="muted">Обоснование: ${justification}.</p>`;
};

// === НГПС ===
const MANDATORY_INDUSTRIES = ['нефтегаз', 'химия', 'энергетика'];
const TECH_BASE = {
  'нефтегаз': 2, 'химия': 2, 'энергетика': 1, 'складГСМ': 1, 'прочее': 1, 'административный': 0
};

function staffRange(numAuto) {
  if (numAuto <= 0) return [8, 10]; // пожарный пост
  const min = 12 * numAuto + 4;     // 3 караула ориентир
  const max = 16 * numAuto + 5;     // 4 караула ориентир
  return [min, max];
}

window.calcNGPS = function () {
  const type = document.getElementById('ngps-type').value;
  const area = +document.getElementById('ngps-area').value || 0;
  const cat = document.getElementById('ngps-cat').value;
  const dist = +document.getElementById('ngps-dist').value || 0;

  const isHazardCat = /^(A|Б|В)$/.test(cat);
  const byList = MANDATORY_INDUSTRIES.includes(type);
  const byRule = (isHazardCat && area >= 3500 && dist > 3);

  let need = false, recommendPost = false, numAuto = 0;
  if (byList) { need = true; numAuto = TECH_BASE[type] || 1; }
  else if (byRule) { need = true; numAuto = TECH_BASE[type] || 1; }
  else if (dist > 3) { recommendPost = true; }

  // Уточнение по крупной площади
  if (need && area > 10000 && numAuto < 2) numAuto = 2;

  let html = '';
  if (need) {
    const [minS, maxS] = staffRange(numAuto);
    const typeHint =
      type === 'нефтегаз' ? ' (АЦ + пенная/порошковая)' :
      type === 'химия' ? ' (АЦ + порошковая)' :
      (numAuto > 0 ? ' (АЦ основная)' : '');

    html = `<p><b>НГПС обязательна:</b> пожарная часть с ${numAuto} авто${typeHint}.</p>
            <p><b>Ориентировочный штат:</b> ${minS}–${maxS} чел (круглосуточно).</p>
            <p class="muted">Обоснование: <a href="#pp1017p8">Перечень №1017, п.8</a>; <a href="#z53p5">Закон «О ГЗ», ст.53</a>.</p>`;
  } else if (recommendPost) {
    const [minS, maxS] = staffRange(0);
    html = `<p><b>НГПС не обязательна.</b> Рекомендуется пожарный пост (без техники): ${minS}–${maxS} чел в штате.</p>
            <p class="muted">Основание: удалённость > 3 км — целесообразно усилить первую реакцию.</p>`;
  } else {
    html = `<p><b>НГПС не обязательна.</b> Достаточно мер общей ПБ и прикрытия ГПС.</p>`;
  }

  document.getElementById('ngps-result').innerHTML = html;
};
